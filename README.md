# Personal Portfolio Website

A responsive portfolio website built using HTML, CSS, and JavaScript to showcase my skills, projects, resume, and contact information.

## Features

- Responsive design for desktop and mobile devices
- About Me section
- Skills showcase
- Projects portfolio
- Downloadable CV
- Contact form powered by Formspree
- Custom Thank You page after form submission

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Formspree
- Vercel

## Live Demo

https://zohaib-portfolio-website.vercel.app/

## Installation

```bash
git clone <repository-url>
```

Open `index.html` in your browser.
